# Example Documentation Page

This is an example page showing various documentation elements.

## Installation

```bash
npm install my-package
```

## Basic Usage

```typescript
import { myFunction } from 'my-package'

const result = myFunction({
  option1: 'value',
  option2: true
})

console.log(result)
```

## Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `option1` | `string` | `'default'` | First configuration option |
| `option2` | `boolean` | `true` | Enable or disable feature |
| `option3` | `number` | `3000` | Timeout in milliseconds |

## Tips

> **Pro Tip:** Use the AI assistant to ask questions about any of these options!

## Code Examples

### JavaScript

```javascript
const { myFunction } = require('my-package')

myFunction({ debug: true })
```

### TypeScript

```typescript
import type { MyConfig } from 'my-package'

const config: MyConfig = {
  option1: 'custom',
  option2: false
}
```
